# Ultrasonic Distance Sensor

Arduino-скетч для измерения расстояния с помощью ультразвукового датчика HC-SR04.

## Описание

Скетч отправляет ультразвуковой импульс, измеряет время возврата эха и выводит расстояние в сантиметрах через Serial Monitor. Обновление каждые 100 мс. Диапазон измерений: 1–300 см.

## Подключение

| HC-SR04 | Arduino |
|---------|---------|
| VCC     | 5V      |
| GND     | GND     |
| Trig    | Pin 10  |
| Echo    | Pin 11  |

## Использование

1. Подключи Arduino к компьютеру
2. Открой скетч в Arduino IDE
3. Выбери плату и порт
4. Загрузи скетч
5. Открой Serial Monitor (скорость: 9600 бод)

## Требования

- Arduino (Uno, Nano или совместимая)
- Датчик HC-SR04
- Arduino IDE
